# Помесячные комментарии по автомобилям

В реестре машин добавлено произвольное поле комментария, например:

```text
Добавлен лимит по согласованию
```

## Поведение

- комментарий хранится отдельно для каждого автомобиля и месяца;
- дата создания и дата последнего изменения устанавливаются автоматически;
- при наступлении нового месяца поле на дашборде становится пустым;
- старые записи не удаляются и остаются в базе как история;
- пустое значение удаляет комментарий выбранного месяца.

## Хранение

Таблица `vehicle_month_comments` создаётся автоматически при запуске API.

Уникальность записи:

```text
plate + year_month
```

## API

Получение комментариев месяца:

```http
GET /vehicle-comments?year_month=2026-07
```

Сохранение или изменение:

```http
PUT /vehicle-comments/33ABC123
Content-Type: application/json

{
  "year_month": "2026-07",
  "comment": "Добавлен лимит по согласованию"
}
```

Очистка комментария:

```http
PUT /vehicle-comments/33ABC123
Content-Type: application/json

{
  "year_month": "2026-07",
  "comment": ""
}
```

## Интерфейс

В таблицу `Реестр машин` автоматически добавляется столбец `Комментарий`. Нажатие на поле открывает редактор. После сохранения рядом с текстом отображается дата последнего изменения.
